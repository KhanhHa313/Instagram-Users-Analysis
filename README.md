# 📊 Instagram User Analysis

📌 Overview:
This project analyzes user engagement patterns across 1.5M+ Instagram user records to identify behavioral trends, cohort differences, and key drivers of platform engagement.

🛠️ Tools & Technologies
- PostgreSQL (SQL)
- Power BI
- Power Query
- Excel

📂 Dataset
- From the Kaggle dataset collection 
- Contains over 1.5 million user records
- Includes account status, login activity, and registration data

🔍 Key Analysis
- Used advanced SQL techniques (CTEs, window functions, Views) to analyse large-scale user data  
- Investigated login frequency and account activity trends  
- Identified factors influencing user engagement  
- Performed cohort analysis based on registration periods and tenure  
- Applied data normalisation techniques to evaluate inactivity patterns  

📈 Key Insights
- Engagement varies significantly across user generation in which younger people present higher engagement score compared to older groups
- Older cohorts demonstrate stronger retention and higher engagement patterns  
- A small percentage of users contribute disproportionately to overall activity  
- Longer time on the platform has a positive correlation with the total engagement score
- Most time on the platform is spent on reactive activities like watching reels or reading posts than higher active counterparts namely posting or commenting
- Most users within the dataset are recently active which is defined as available on the platform within the last 170 days on average
